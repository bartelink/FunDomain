FunDomain
=========
This is a WIP push of a spike of [NEventStore](https://github.com/NEventStore/NEventStore) integration for 
[Jérémie Chassaing](http://twitter.com/thinkb4coding)'s [FsUno](https://github.com/thinkbeforecoding/FsUno) 
Functional Event Sourcing sample.

While some of the test style you see here may eventually make it into FsUno itself in some form, this should be 
considered a fork that'll expire in due course.
